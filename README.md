\#Budget Bites



\*\*Find Affordable Restaurants That Fit Your Budget\*\*



Budget Bites is a Progressive Web Application (PWA) designed to help students and budget-conscious individuals discover affordable restaurants based on their specific budget constraints and cuisine preferences.



---



\## Project Information



\- \*\*Type:\*\* Final Project for DAW (Desarrollo de Aplicaciones Web)

\- \*\*Academic Year:\*\* 2025-2026

\- \*\*Author:\*\* \Mohammed Abdelbar

\- \*\*Deadline:\*\* May 1, 2025



---



\##  Features



\-  Search restaurants by budget and cuisine type

\-  Save favorite restaurants

\-  AI-powered restaurant recommendations

\-  Progressive Web App (installable on mobile)

\-  Secure user authentication

\-  User role management (Admin, User, Guest)



---



\##  Technology Stack



\### Frontend

\- React 18+

\- HTML5, CSS3, JavaScript ES6+

\- PWA (Service Worker, Web App Manifest)



\### Backend

\- \*\*PHP 8+\*\* (Main REST API)

\- \*\*Python 3.10+\*\* (AI Recommendation Microservice)



\### Database

\- MySQL 8.0+



\### External APIs

\- Yelp Fusion API (Restaurant data)

\- OpenAI/Claude API (AI recommendations)
